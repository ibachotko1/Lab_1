# WarehouseManagement

WarehouseManagement - это система управления складскими операциями с веб-интерфейсом и бизнес-логикой на .NET.

### 📋 Описание
- Система обеспечивает полный цикл управления складскими операциями: приемку товаров, списание, инвентаризацию и отслеживание остатков. Реализована с разделением на backend и frontend с соблюдением бизнес-правил и инвариантов.

### 🧩 Структура решения:
```
WarehouseManagement/
│
├── Warehouse.Back/ # Backend проект
│ ├── Models/ # Модели данных
│ │ ├── Product.cs # Товар
│ │ ├── OperationRecord.cs # Запись операции
│ │ └── OperationType.cs # Тип операции
│ │
│ ├── Repository/ # Слой доступа к данным
│ │ └── WarehouseRepository.cs
│ │
│ ├── Service/ # Бизнес-логика
│ │ └── WarehouseService.cs
│ │
│ └── Interface/ # Контракты
│ ├── IWarehouseRepository.cs
│ └── IWarehouseService.cs
│
├── Warehouse.Front/ # Frontend проект
│ ├── Pages/ # Веб-страницы
│ │ ├── InventoryPage.xaml # Инвентаризация
│ │ ├── OperationsPage.xaml # Журнал операций
│ │ ├── ProductsPage.xaml # Список товаров
│ │ ├── ReceiveGoodsPage.xaml # Приемка товара
│ │ └── WriteOfPage.xaml # Списание товара
│ │
│ └── Services/ # Клиентские сервисы
│ └── WarehouseConnector.cs
│
└── Warehouse.Tests/ # Модульные тесты
```

### ✨ Основные возможности
- 📥 Приемка товаров на склад с валидацией данных
- 📤 Списание товаров со склада с контролем остатков
- 📊 Проведение инвентаризации и корректировка остатков
- 📋 Просмотр журнала операций
- 💰 Автоматический расчет стоимости инвентаря
- 🔍 Управление товарами через уникальные SKU
- ⚠️ Обработка исключений и бизнес-правил

## 🚀 Установка и запуск
### Системные требования
- .NET 6.0 или выше
- Windows 10/11 или Linux/macOS
